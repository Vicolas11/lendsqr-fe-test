# Lendsqr FE Test
Lendsqr Frontend Test app build with React framework using TypeScript programming language.

## About
The app basically fetches data from a mock api and renders it on a table where each individual users' profile can be viewed.

## Getting Started
1. git clone
2. yarn install (to install all the dependencies)
3. yarn start

## Usage
Creat a .env and then paste the mock api there setting variable name REACT_APP_MOCK_API='mockapi link'
